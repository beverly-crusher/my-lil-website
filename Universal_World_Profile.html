Universal World Profile

A Universal World Profile is a code describing the characteristics of a planet.
They are read as follows:

A-X 0-A 0-F 0-A 0-C 0-F 0-F - 0-_
Starport quality Planet size Atmosphere type Hydrographic percentage Population Government type Law level - Tech Level

with A through F corresponding to values 10 through 15

The UWP is followed by codes for any imperial bases present, any trade codes, then any travel codes.

For description of the corresponding values, please see the Traveller Core Rulebook, p. 214-228 (ed. 1) or p. 246-261 (ed. 2).

example:

Kingfisher D6585A9-10 S A
